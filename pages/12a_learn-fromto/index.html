Belajar gsap.fromTo()

Method gsap.fromTo() dipake buat animasikan elemen dari state baru ke state baru.

Method gsap.fromTo() mirip dengan method gsap.from() dan gsap.to(), tapi bedanya adalah gsap.fromTo() animasikan elemen dari state baru ke state baru, sementara gsap.from() animasikan elemen dari state baru ke state saat ini, dan gsap.to() animasikan elemen dari state saat ini ke state baru.
