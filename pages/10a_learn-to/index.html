Belajar gsap.to()

Method gsap.to() dipake buat animasikan elemen dari state saat ini ke state baru.

Method gsap.to() mirip dengan method gsap.from(), tapi bedanya adalah gsap.to() animasikan elemen dari state saat ini ke state baru, sementara gsap.from() animasikan elemen dari state baru ke state saat ini.
