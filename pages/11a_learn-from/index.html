Belajar gsap.from()

Method gsap.from() dipake buat animasikan elemen dari state baru ke state saat ini.

Method gsap.from() mirip dengan method gsap.to(), tapi bedanya adalah gsap.from() animasikan elemen dari state baru ke state saat ini, sementara gsap.to() animasikan elemen dari state saat ini ke state baru.
